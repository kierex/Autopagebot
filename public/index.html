🤖 AutoPageBot Setup Tutorial

Complete guide to creating and deploying your Facebook Messenger bot

📋
Prerequisites
  • A Facebook account
  • A Powered by: developer.facebook.com
1
Initial Facebook Setup
  1. Log in to Facebook
  2. Create a Facebook Page
  3. Go to developers.facebook.com/apps
2
Create a Facebook App
  1. Click "Create App"
  2. Select "Business" as type
  3. Fill in details and click "Create App"
3
Generate Access Token
  1. Add Messenger product
  2. Connect your Facebook Page
  3. Generate and copy Page Access Token
⚠️ Save this token securely!
4
Connect Page
  1. Go to Connect Page section
  2. Enter name and paste token
  3. Click Connect Page
5
Set Up Webhooks
  1. Go to Messenger API Settings
  2. Add Callback URL: loading...
  3. Verify Token: bot
  4. Subscribe to messages, messaging_postbacks
6
Make App Live
  1. Toggle app from Development to Live
  2. Generate new token and reconnect
✅ Your bot is now live!
